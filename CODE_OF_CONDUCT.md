# Code Of Conduct

This project follows a simple standard: be respectful, precise, and collaborative.

## Expected Behavior

- use welcoming and professional language
- focus criticism on ideas, code, and behavior
- assume good intent while keeping technical discussion rigorous
- help keep issue threads and pull requests actionable

## Unacceptable Behavior

- harassment, intimidation, or personal attacks
- discriminatory language or exclusionary behavior
- deliberate disruption, spam, or bad-faith escalation

## Reporting

If you encounter behavior that violates this code of conduct, open a private security-style report or contact the maintainers through the repository's private channels once they are configured.
