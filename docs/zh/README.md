# 中文文档

`paperfetch` 是一个面向开发者和 AI agent 的论文元数据提取工具。输入可以是 `arXiv ID`、`arXiv URL` 或 `alphaXiv URL`，输出是统一的 JSON 结构和 Markdown dossier。

## 建议阅读顺序

- [快速开始](getting-started.md)
- [架构说明](architecture.md)
- [常见问题](faq.md)
- [English docs index](../README.md)

## 仓库定位

- 用 `arXiv` 作为规范元数据来源
- 用 `alphaXiv` 作为可选增强层
- 对 agent、自动化脚本、轻量数据流程友好
